# API GestPFE

API Laravel du projet GestPFE ENSA Marrakech. Consultez le fichier `../README.md` pour l'installation, la configuration OAuth, les comptes de démonstration et les commandes de test.

Les routes sont déclarées dans `routes/api.php`. Toutes les routes métier sont protégées par `auth.api`, une limite de débit et un middleware RBAC.
